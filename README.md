# Web DEV Project
 
📝 Task Management Website

A simple and interactive Task Management Web Application built using HTML, CSS, and JavaScript.
This project allows users to organize and track their tasks efficiently using different categories.

---

🚀 Features

📌 Task Categories

Tasks can be organized into three stages:

- To Start
- In Progress
- Completed

⚙️ Functionalities

- ➕ Create new tasks
- 🔄 Move tasks between categories
- 🗑️ Delete tasks
- 📅 Set deadlines for tasks
- 💾 Data stored locally using localStorage
- 🌗 Light Mode & Dark Mode toggle

---

🎨 UI Design

Light Mode

- Tomato
- Light Blue
- White

Dark Mode

- Black
- Dark Blue
- Orange

---

🛠️ Technologies Used

- HTML – Structure
- CSS – Styling & themes
- JavaScript – Functionality & logic
- LocalStorage – Data persistence

▶️ How to Run

1. Download or clone this repository
2. Open the folder
3. Double-click "index.html"
4. The website will open in your browser
